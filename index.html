M.A.S.S. (Mutual Aid & Shared Support)

User registers with phone number, allows user to be as anonymous as possible. Logged in user is able to submit a list of things needed as well as a location of where the trusted admin is to drop off the requested items.
Admin is able to see a list of orders with the items requested from all users.

Front-end: JavaScript, React, HTML5, CSS3

Back-end: JavaScript, Node.js, Express.js, PostgreSQL

Live App | Client Repo | Server Repo

Spaced Repetition

This project uses spaced repetition to teach Italian. The user guesses the translation for the Italian word and the next page shows if they got the word right or wrong. For each user, score is kept on how many times they have guessed on the word, the correct and incorrect number of guesses on each word.

Front-end: JavaScript, React, HTML5, CSS3

Back-end: JavaScript, Node.js, Express.js, PostgreSQL

Live App | Client Repo | Server Repo
